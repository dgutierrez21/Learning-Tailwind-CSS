Min Width
Utilidades para establecer el ancho mínimo de un elemento.

Clase  |  Propiedades
min-w-0  |  min-width: 0px;
min-w-full  |  min-width: 100%;
min-w-min  |  min-width: min-content;
min-w-max  |  min-width: max-content;
min-w-fit  |  min-width: fit-content;

Establecer el ancho mínimo
Establezca el ancho mínimo de un elemento utilizando las utilidades min-w-{width}.

<span class="min-w-full ...">
  <!-- ... -->
</span>

...

Valores arbitrarios
Si necesita usar un valor de min-width único que no tiene sentido incluir en su tema, use corchetes para generar una propiedad sobre la marcha utilizando cualquier valor arbitrario.

<div class="min-w-[50%]">
  <!-- ... -->
</div>
