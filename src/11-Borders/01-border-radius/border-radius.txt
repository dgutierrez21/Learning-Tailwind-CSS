Border Radius
Utilidades para controlar el radio de borde de un elemento.

Clase  |  Propiedades
rounded-none  |  border-radius: 0px;
rounded-sm  |  border-radius: 0.125rem; /* 2px */
rounded  |  border-radius: 0.25rem; /* 4px */
rounded-md  |  border-radius: 0.375rem; /* 6px */
rounded-lg  |  border-radius: 0.5rem; /* 8px */
rounded-xl  |  border-radius: 0.75rem; /* 12px */
rounded-2xl  |  border-radius: 1rem; /* 16px */
rounded-3xl  |  border-radius: 1.5rem; /* 24px */
rounded-full  |  border-radius: 9999px;

Referencia completa en https://tailwindcss.com/docs/border-radius

Esquinas redondeadas
Utilice utilidades como .rounded-sm, .rounded o .rounded-lg para aplicar diferentes tamaños de radio de borde a un elemento.

Botones de píldora

Sin redondeo
Utilice rounded-none para quitar un radio de borde existente de un elemento.

Esto se usa más comúnmente para eliminar un radio de borde que se aplicó en un punto de interrupción más pequeño.

...

Redondear los lados por separado
Usa rounded-{t|r|b|l}{-size?} solo un lado de un elemento.

...

Redondear esquinas por separado
Use rounded-{tl|tr|br|bl}{-size?} solo una esquina de un elemento.
