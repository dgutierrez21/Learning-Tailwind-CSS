Object Fit
Utilidades para controlar cómo se debe cambiar el tamaño del contenido de un elemento reemplazado.

Clase  |  Propiedades
object-contain  |  object-fit: contain;
object-cover  |  object-fit: cover;
object-fill  |  object-fit: fill;
object-none  |  object-fit: none;
object-scale-down  |  object-fit: scale-down;

Cambio de tamaño para cubrir un contenedor
Cambiar el tamaño del contenido de un elemento para cubrir su contenedor mediante .object-cover.

...

Contener un elemento
Cambiar el tamaño del contenido de un elemento para que permanezca contenido dentro de su contenedor mediante .object-contain.

...

Estiramiento para adaptarse a un contenedor
Estirar el contenido de un elemento para que se ajuste a su contenedor mediante .object-fill.

...

Reducción de escala si es demasiado grande
Mostrar el contenido de un elemento en su tamaño original, pero reducirlo para que se ajuste a su contenedor si es necesario mediante .object-scale-down.

...

Usar el tamaño original de un elemento
Mostrar el contenido de un elemento en su tamaño original ignorando el tamaño del contenedor mediante .object-none.
