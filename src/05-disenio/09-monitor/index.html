Al controlar el flujo de texto, utilizando la propiedad CSS display: inline hará que el texto dentro del elemento se ajuste normalmente.
Durante el uso de la propiedad display: inline-block ajustará el elemento para evitar que el texto se extienda más allá de su elemento primario.
Por último, utilizando la propiedad display: block pondrá el elemento en su propia línea y llenará su padre.
