Grid Auto Flow
Utilidades para controlar cómo se colocan automáticamente los elementos de una cuadrícula.

Clase  |  Propiedades
grid-flow-row  |  grid-auto-flow: row;
grid-flow-col  |  grid-auto-flow: column;
grid-flow-dense  |  grid-auto-flow: dense;
grid-flow-row-dense  |  grid-auto-flow: row dense;
grid-flow-col-dense  |  grid-auto-flow: column dense;

Control de la colocación de elementos de cuadrícula
Utilice las utilidades grid-flow-{keyword} para controlar cómo funciona el algoritmo de colocación automática para un diseño de cuadrícula.
